Bank RD Account Interest Calculator (Java)

A console-based Java application that calculates the maturity amount of a Recurring Deposit (RD) account based on customer age, gender, tenure, and monthly deposit.

The program validates user input using custom exceptions and applies different interest rates according to bank rules.

Features

Input validation using custom exception

Interest rate based on age & gender

Calculates:

Total deposited amount

Interest earned

Maturity amount

Supports multiple test cases
