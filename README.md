Train Reservation System (Java)

A console-based Train Reservation System built using Core Java and OOP concepts.
The application allows passengers to book tickets, cancel reservations, and view booking history while handling errors using custom exceptions.

 Features

Passenger validation

Book train ticket

Cancel reservation

View passenger reservation history

Automatic seat availability update

Fare calculation

Custom exception handling
